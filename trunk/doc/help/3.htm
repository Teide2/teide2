0CREACIÓN DE LA BASE DE DATOS (.XLS)

 

La base de datos tiene que tener como mínimo 4 tablas u hojas (variables, microdatos, edits y missing).

Hoja INFO_MISSING (): contiene los valores missing a tener en cuenta y el valor numérico que toman. Como máximo serán 4 y como mínimo 1. Debe tener obligatoriamente el valor No Procede, el cual se puede representar de  la siguiente manera: No_Procede ó NO_PROCEDE ó NoProcede.

Hoja VARIABLES (): debe tener como mínimo los siguientes campos.

ID: identificador de cada variables.

NOMBRE: nombre de la variable.

INFO_VARIABLE: información sobre la variable.

RANGO: representa los valores que puede tomar la variable.

            desde:hasta    ó     valor1, valor2, valor2, etc

FILTRO: indica el filtro que tiene la variable.

INFO_FILTRO: explica en qué consiste el filtro.

SENTIDO_FILTRO (opcional): indica en qué sentido se tomará el filtro.

      ‘a’: if (nofiltro) then valor=NP

      ‘b’: if (filtro) then valor!=NP

      ‘c’: a y b

       Este campo es opcional en el caso de que no se ponga el valor por defecto será ‘a’.

IMPUTABLE: indica si la variable se puede imputar. Es 1/0.

TIPO: indica el tipo de la variable (0: Continuo;  1: Discreto en lista; 2: Discreto en rango; 3: ignorable).

PESO: el peso que tiene la variable.

MAPPING: indica la tabla mapping que contiene la descripción sobre los valores que toma esta variable.

IMP_NUM: indica si la variable será imputada siguiendo una fórmula (MEDIANA, MEDIA_R, MEDIA ó fórmula de regresión), en otro caso será mediante registro donante.

Missing incluidos en la tabla anterior. Se pondrán tantos campos y con el mismo nombre como se hayan puesto en la tabla de missing. Es 1/0.
